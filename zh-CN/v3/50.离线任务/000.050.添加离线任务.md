#  添加离线任务

> 添加任务需要提供上一次的 hash，此接口支持批量操作

> 提交需要的参数

|  参数  	|  类型  	| 可选 	|           说明           	|
|:------:	|:------:	|:----:	|:------------------------:	|
|  task  	| array[TaskInfo] 	|  否  	|  需要添加的任务  	|
|  savePath  	| string 	|  是  	|        目的路径id，与 saveIdentity 二选一    	|
| saveIdentity 	| string 	|  是  	|  目的父目录的id，与 savePath 二选一	|

> TaskInfo

|  参数  	|  类型  	| 可选 	|           说明           	|
|:------:	|:------:	|:----:	|:------------------------:	|
|  hash  	| string 	|  否  	|   上一步解析出的哈希  	|
|  ignore  	| array[string] 	|  是  	|    需要忽略的文件ID 	|
