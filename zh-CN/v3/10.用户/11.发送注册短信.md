#### 发送注册短信

注册用户需要用户有一个手机号以便接收短消息。此手机号用来识别用户。

* 发送请求

```
POST
```
```
/v3/user/sendMessage/register
```

```json
{
	"countryCode": "86",
	"phone": "17771737760"
}
```

* 如果用户已经注册用户，则会报错

```json
{
    "success": false,
    "status": 422,
    "refrence": "PHONE_ALREADY_EXISTS",
    "message": "该电话号码已经注册了用户"
}
```
