# 删除回收站项目

> 将回收站中具体项目删除

#### 复制文件或文件夹使用的参数

|  参数  	|  类型  	| 可选 	|           说明           	|
|:------:	|:------:	|:----:	|:------------------------:	|
|  sourceIdentity  	| array<string> 	|  是  	|  回收id的数组 	|


```POST``` ```/v3/trash/delete```


```json
{
	"sourceIdentity":["103823a3239487ef28338e221"],
}
```




#### 注意

* 异步操作

#### 响应

> 删除成功

```json
{
    "successCount": 1
}
```
