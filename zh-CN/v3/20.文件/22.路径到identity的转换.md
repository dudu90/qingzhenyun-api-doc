#### 路径转identity

* 示范如何把路径 如 /a/b/c/d 转化为 identity，在进行某些操作的时候相对比较简单

```
nj/lo/./ju09//0v1/
```

1. 获取路径，去除首尾空格，路径中的../ ./ 等

```
nj/lo/ju09/0v1/
```

2. 如果以 / 结尾 则去掉

```
nj/lo/ju09/0v1
```

3. 如果起始 不是 / 则加上 /
```
/nj/lo/ju09/0v1
```

4. 取其 32位 md5 小写，则为 identity

```
18b5258ddfa9e34e3debf358635a9bdd
```
